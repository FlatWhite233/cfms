## ✨功能概述

<br>

### 管理员

- 登录（密码MD5加密）
- 管理班级
- 管理学生
- 设置班级管理员

### 班级管理员

- 查询班费变更
- 确认变更
- 查询班费余额

### 学生

- 查询班费变更（只能查到本班记录）
- 提交班费变更
- 查询班费余额
